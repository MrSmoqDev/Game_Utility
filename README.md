## 📖 About

**Game Utility** is a powerful, open-source memory manipulation framework designed for game modification and training purposes. Built with Python and utilizing direct NT syscall operations, it provides a flexible platform for creating custom game trainers with real-time memory editing capabilities.

### ⚡ Key Features

-  **Direct Memory Access** - Uses NT syscalls (`NtReadVirtualMemory`/`NtWriteVirtualMemory`) to bypass standard API hooks
- 🎯 **Multi-Game Support** - Dynamic configuration system via JSON files
- 🔧 **Flexible Cheat System** - Support for pointer chains, static addresses, and dynamic values
- 📡 **GitHub Integration** - Automatic trainer updates and configuration fetching
- 🎨 **Console UI** - Clean, intuitive terminal-based interface
- 🔐 **Process Isolation** - Each cheat runs in isolated threads with error handling
- 📊 **Real-time Monitoring** - Live memory value tracking and debugging
- 🎮 **Group Cheats** - Link multiple cheats to activate simultaneously
