## 📖 About

**Game Utility** is a powerful, open-source tool that allows you to modify game values in real-time (like health, ammo, or currency). It features a clean menu, automatic updates, and support for multiple games through a simple configuration system.

### ⚡ Key Features

- 🎯 **Multi-Game Support**: Works with any game that doesn't use kernel-level anti-cheat.
- 🔄 **Auto-Updates**: Automatically fetches the latest game configurations without downloading them.
- ⚡ **Real-Time Editing**: Instantly apply changes like Godmode or Infinite Ammo.
- 🛡️ **Stable & Safe**: Uses advanced memory access methods to ensure stability.

## ⬇️ Installation

1. **Download** the latest release from the [Releases](https://github.com/MrSmoqDev/Game_Utility/releases) page.
2. **Extract** the downloaded `.zip` file to a folder of your choice.
3. **Run as Administrator**: Right-click `Game_Utility.exe` and select **"Run as administrator"**.  
   *(⚠️ This is strictly required for the tool to access game memory).*
